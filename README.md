# Банк задач ЕГЭ на Django

Проект на Django для тренировки решения задач ЕГЭ по математике.

## Функциональность

- главная страница;
- каталог заданий;
- страница отдельного задания;
- проверка ответа пользователя;
- вывод пояснения к решению;
- загрузка изображения к условию и пояснению;
- список тем;
- фильтрация по теме;
- фильтрация по сложности;
- поиск по заданиям;
- административная панель Django для управления заданиями.

## Стек

- Python
- Django
- SQLite
- HTML
- CSS

## Основные страницы

- `/` — главная страница;
- `/tasks/` — каталог заданий;
- `/tasks/<id>/` — страница отдельного задания;
- `/topics/` — список тем;
- `/topics/<id>/` — задания по выбранной теме;
- `/admin/` — административная панель.

## Демо-данные

В репозитории есть fixture с демонстрационными темами и заданиями.

После выполнения миграций можно загрузить демо-данные:
